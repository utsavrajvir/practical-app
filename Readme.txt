Command to exectue before running
=> npm i

Command to run project 
=> npm start

About project
=> Need to register user from registration page
=> You can login with that credentials.
=> Able to create as many users you want with different username.
=> Each user  data will be maintained based on it's update, delete or create.
=> Notification for birthday greetings will also be maintained for every user.

Dependency
=> react-redux
=> react-router-dom
=> redux-thunk
=> antd
